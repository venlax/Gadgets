## This is a translator powered by googletrans

### dependency
```shell
pip install googletrans==4.0.0-rc1
```

```shell
Usage: 
    Linux/MacOS:  python3 main.py [OPTIONS] [ARGUMENTS]  
    Windows: python3.exe main.py [OPTIONS] [ARGUMENTS]

Options:
  -m, --mode MODE         Specify the mode: file or text.
  -f, --file FILENAME     Specify the input file.
  -o, --output FILENAME   Specify the output file.
  -t, --text TEXT         Specity the text needed translated.
  -s, --src  src_lang     The source language, default: auto.
  -d, --dest dest_lang    The dest language, default: en. 

```
